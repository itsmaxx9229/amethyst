## About

Amethyst is a lightweight Fabric modpack focused on performance optimization and visual polish. It doesn't change Minecraft's core gameplay; it just makes everything look, feel, and run better. Smoother animations, better UI, shader support, and a collection of carefully picked resource packs to refine the vanilla experience.

---

## What's Included

**Performance**
Sodium, Lithium, Iris, FerriteCore, Krypton, EntityCulling, ImmediatelyFast and more; squeezing out every bit of performance Fabric has to offer.

**Visual Polish**
3D Skin Layers, Not Enough Animations, Continuity, smooth scrolling, smooth GUI transitions, and Zoomify for that extra visual finesse.

**Quality of Life**
Inventory Profiles Next, Mouse Tweaks, Better F3, More Chat History, Camera Tweaks, Mod Menu and other small but meaningful improvements to everyday gameplay.

**Resource Packs**
A curated set of resource packs including Better Lanterns, Bushier Bushes, 3D Crops, 3D Ladders, 3D Rails, Fresh Flower Pots, Low On Fire, Vanilla Tweaks and more.
